# 🗜 Архивация и выполнение (`zip` и `cmd`)

## 🗜 `zip`: Создание архивов

Создает стандартный архив `.zip`. В отличие от базовых архиваторов, которые "сплющивают" директории, `fflow` сохраняет точную относительную структуру папок, используя `filepath.ToSlash` для кроссплатформенной совместимости.

**Пример:**
```bash
fflow zip -r -e .go,.mod -o source_backup.zip
```

## ⚡ `cmd`: Выполнение Shell-команд

Выполняет кастомную shell-команду для **каждого найденного файла** параллельно. Это невероятно мощный инструмент для пакетной обработки изображений, компиляции кода или запуска линтеров.

**Плейсхолдер `{}`:**
Используйте `{}` в строке `--exec`. `fflow` заменит его на **абсолютный путь** к текущему файлу, корректно обернув в кавычки для обработки пробелов в именах.

**Кроссплатформенное выполнение:**
- В **Linux/macOS** команда оборачивается в `sh -c`.
- В **Windows** команда оборачивается в `cmd /c`.

**Примеры:**

**1. Конвертация всех PNG в JPG с помощью ImageMagick:**
```bash
fflow cmd -e .png --exec "magick {} {}.jpg"
```

**2. Запуск Go-линтера на всех Go-файлах:**
```bash
fflow cmd -r -e .go --exec "golangci-lint run {}"
```

**3. Распаковка всех ZIP-файлов в директории:**
```bash
fflow cmd -e .zip --exec "unzip {} -d extracted/"
```
