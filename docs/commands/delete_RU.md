# 🗑 Безопасное удаление (`delete`)

Команда `delete` безвозвратно удаляет файлы, соответствующие вашим фильтрам. Она выполняется параллельно для мгновенной обработки больших пакетов.

⚠️ **ПРЕДУПРЕЖДЕНИЕ**: Это действие необратимо. Всегда тестируйте свои фильтры с помощью команды `info`, чтобы убедиться, что вы выбрали правильные файлы!

## 💻 Примеры использования

**Удаление всех временных файлов сборки:**
```bash
fflow delete -r -e .tmp,.bak,.log
```

**Удаление файлов больше 5 ГБ в папке Загрузки:**
```bash
fflow delete ~/Downloads --min-size 5gb
```

**Альтернатива Dry-Run (Тестовому запуску):**
Поскольку в `fflow` нет встроенного флага `--dry-run` для удаления, используйте `info` для предпросмотра:
```bash
fflow info -r -e .tmp,.bak,.log
```
