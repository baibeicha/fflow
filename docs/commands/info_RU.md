# 📂 Анализ директорий (`info`)

Команда `info` — это высокопроизводительный анализатор директорий. В отличие от стандартных утилит, требующих нескольких проходов для расчета размеров папок, `fflow` использует **алгоритм восходящего накопления (bottom-up) за один проход**.

## 🧠 Как это работает
1. **Обход за один проход**: Использует `fastwalk` для чтения каждого файла и директории ровно один раз.
2. **Накопление размеров**: По мере нахождения файлов их размер прибавляется к их непосредственной родительской директории в карте памяти.
3. **Восходящее распространение**: Директории сортируются по глубине пути (от самых глубоких к корню), и размеры распространяются вверх по дереву. Это гарантирует 100% точные размеры вложенных папок без избыточных системных вызовов `stat()`.

## 💻 Примеры использования

**Анализ текущей директории (без рекурсии):**
```bash
fflow info .
```

**Поиск самых тяжелых директорий в проекте рекурсивно:**
```bash
fflow info ./project -r --sort-by size:desc
```

**Анализ только конкретных типов файлов (например, медиа):**
```bash
fflow info ./media -r -e .mp4,.mkv,.avi --min-size 1gb
```

## 📊 Формат вывода
`info` выводит красиво отформатированную таблицу с использованием `tabwriter`, отображая:
- **Имя** (Директории помечаются слэшем `/` на конце)
- **Размер** (Человекочитаемый формат, например, `1.4 GB`)
- **Время изменения** (`YYYY-MM-DD HH:MM:SS`)
- **Относительный путь**
