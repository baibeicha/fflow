# 📄 Руководство по генерации PDF

Команда `fflow pdf` — это мощнейший инструмент для создания документов. Она читает файлы, категоризирует их (Код, Текст, Markdown, Таблица, Изображение) и рендерит их в связный PDF-документ, используя библиотеку `go-pdf/fpdf`.

## 🧠 Как это работает
1. **Сбор файлов**: Собирает файлы на основе ваших глобальных фильтров (`-e`, `-r`, `--min-size` и т.д.).
2. **Категоризация**: Маппит расширения на типы (например, `.go` -> Код, `.csv` -> Таблица, `.md` -> Markdown).
3. **Параллельное чтение**: Читает содержимое файлов конкурентно, используя пулы воркеров для максимизации пропускной способности I/O.
4. **Рендеринг**: Применяет выбранный Пресет или Кастомную Конфигурацию для отрисовки страниц, автоматически обрабатывая разрывы страниц, подписи и типографику.

## 🎨 Пресеты и локализация
Пресеты определены в `pkg/files/pdf/presets`. Каждый пресет содержит конфигурации `EN` и `RU`.
*Пример: Пресет `AcademicReport` на русском языке автоматически будет использовать "Таблица - " для подписей таблиц и "Рисунок - " для изображений, в то время как английская версия использует "Table - " и "Figure - ".*

## ⚙️ Полный справочник флагов

### Общий макет
| Флаг | Описание | По умолчанию |
| :--- | :--- | :--- |
| `--preset` | Выбрать встроенный шаблон. | `A4Portrait` |
| `--lang` | Локализация подписей (`en`, `ru`). | `ru` |
| `--output`, `-o` | Путь к выходному PDF-файлу. | `output.pdf` |
| `--orientation` | `portrait` или `landscape`. | Из пресета |
| `--page-size` | `A4`, `A3`, `A5`. | Из пресета |
| `--margin` | Поля страницы в мм. | Из пресета |
| `--font-name` | Базовый шрифт. | `Arial` |
| `--font-size` | Размер базового шрифта в pt. | `10.0` |
| `--font-path` | Путь к кастомному файлу шрифта `.ttf`. | Пусто (Использует встроенный Arial) |

### 📊 Конфигурация таблиц (CSV/TSV/PSV)
| Флаг | Описание |
| :--- | :--- |
| `--table-separator` | Принудительный разделитель (`,` , `;`, `\t`). Авто-определение, если пусто. |
| `--table-enable-caption` | Добавить подпись над/под таблицей. |
| `--table-caption-position` | `top` (сверху) или `bottom` (снизу). |
| `--table-caption-alignment`| `left`, `center`, `right`. |
| `--table-caption-margin` | Отступ между подписью и таблицей (мм). |
| `--table-caption-prefix` | Текст перед именем файла (напр., "Таблица 1: "). |
| `--table-stretch-width` | Растянуть таблицу на всю ширину страницы. |

### 🖼 Конфигурация изображений
| Флаг | Описание |
| :--- | :--- |
| `--image-enable-caption` | Добавить подпись с именем файла. |
| `--image-caption-position` | `top` или `bottom`. |
| `--image-caption-alignment`| `left`, `center`, `right`. |
| `--image-caption-margin` | Отступ между картинкой и подписью. |
| `--image-caption-prefix` | Текстовый префикс (напр., "Рис. "). |

### 💻 Конфигурация кода
| Флаг | Описание |
| :--- | :--- |
| `--code-style-as-block` | Рендерить с фоновым цветом (блоком). |
| `--code-disable-header` | Скрыть верхнюю панель с именем файла/языком. |
| `--code-font-name` | Моноширинный шрифт (По умолчанию: `Courier`). |
| `--code-font-size` | Размер шрифта кода. |
| `--code-bg-color` | RGB цвет фона (напр., `245,246,248`). |
| `--code-text-color` | RGB цвет текста (напр., `36,41,47`). |

### 📝 Конфигурация Текста и Markdown
| Флаг | Описание |
| :--- | :--- |
| `--text-render-markdown` | Принудительно парсить Markdown для ВСЕХ `.txt` файлов. |

## 💡 Pro-Советы
- **Удаление эмодзи**: `fflow` автоматически удаляет эмодзи из текста и кода, чтобы предотвратить ошибки кодирования PDF при использовании стандартных шрифтов.
- **Обработка BOM**: Парсер CSV автоматически обнаруживает и пропускает UTF-8 BOM (Byte Order Mark).
- **Большие файлы**: Файлы больше 10 МБ читаются последовательно во время рендеринга, а не загружаются в RAM целиком, что предотвращает краши OOM (Out of Memory).
