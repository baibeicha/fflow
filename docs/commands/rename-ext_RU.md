# ✏️ Массовое переименование (`rename` и `ext`)

Эти команды изменяют имена и расширения файлов. Как и движок передачи, они включают встроенную защиту от коллизий, чтобы предотвратить случайную перезапись в процессе переименования.

## 🔄 `rename`: Поиск, замена, префиксы и суффиксы

Изменяет базовое имя файла (игнорируя расширение).

**Флаги:**
- `--search`: Строка для поиска в имени файла.
- `--replace`: Строка для замены найденного.
- `--prefix`: Строка для добавления в начало имени.
- `--suffix`: Строка для добавления в конец имени (перед расширением).

**Пример: Очистка имен файлов**
```bash
# Заменяет пробелы на подчеркивания и добавляет префикс с датой
fflow rename -r -e .jpg --search " " --replace "_" --prefix "2026_"
```

## 📎 `ext`: Смена расширений

Меняет расширения файлов пакетами. Вам не нужно указывать точку (`.`); `fflow` добавит её автоматически, если она отсутствует.

**Пример: Конвертация всех CSV в TSV**
```bash
fflow ext -r -e .csv --to tsv
```
